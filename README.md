# Currency Converter

A simple and responsive Currency Converter web application built using HTML, CSS, and JavaScript. It allows users to convert currencies in real time using a free exchange rate API.

## Features

- Real-time currency conversion
- Supports multiple international currencies
- Country flags for selected currencies
- Responsive and modern UI
- Default currency conversion from USD to INR
- User-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Exchange Rate API
- Flags API

## How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/saurabh-2414/currency-converter
```

2. Open the project folder in Visual Studio Code.

3. Install the Live Server extension (if not already installed).

4. Right-click on `ind.html` and select:

```
Open with Live Server
```

5. The application will open in your browser.
